# APK 전달 및 설치 안내

## 1. APK 전달 방식

APK 파일은 다음 방식으로 전달할 수 있다.

- 카카오톡 파일 전송
- Google Drive 공유
- OneDrive 공유
- USB 케이블 복사
- 메신저 파일 전송

## 2. 설치 전 준비

Android 기기에서 APK를 직접 설치하려면 "알 수 없는 앱 설치" 권한이 필요할 수 있다.

일반적인 경로:

```text
설정 -> 보안 또는 앱 -> 알 수 없는 앱 설치 -> 파일을 여는 앱 허용
```

기기 제조사와 Android 버전에 따라 메뉴 이름은 다를 수 있다.

## 3. 업데이트 방식

스토어 배포가 아니므로 업데이트는 새 APK를 다시 전달해서 설치한다.

같은 앱으로 업데이트하려면 다음 조건을 지켜야 한다.

- 앱 패키지명이 같아야 한다.
- 릴리즈 서명 키가 같아야 한다.
- 버전 코드가 이전 APK보다 높아야 한다.

## 4. APK 파일 위치

Flutter 릴리즈 빌드 후 APK는 보통 아래 위치에 생성된다.

```text
build/app/outputs/flutter-apk/app-release.apk
```

## 5. 설치 테스트 체크리스트

- APK 설치가 완료되는가
- 앱 실행이 되는가
- 곡 목록이 표시되는가
- 음악 재생이 되는가
- 화면을 꺼도 재생이 유지되는가
- 알림창에서 재생/일시정지가 되는가
