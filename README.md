# 도휘 플레이어

도휘를 위한 Android 음악 플레이어입니다.

## 다운로드

[도휘 플레이어 APK 다운로드](https://github.com/Hansoo3366/dohwi-player/releases/latest)

Release 페이지의 **Assets**에서 `도휘 플레이어.apk`를 받으면 됩니다.

## 설치

1. APK 파일을 Android 기기에 다운로드합니다.
2. 파일을 열어 설치합니다.
3. 설치가 막히면 Android 설정에서 **알 수 없는 앱 설치 허용**을 켭니다.

## 참고

이 앱은 Google Play Store 배포용이 아니라 APK 직접 설치용입니다.
